Mysql数据库连接与查询

用PHP查询Mysql数据库

打开Navicat for MySQL, 新建一个连接，如mybook，在里面建立表，点击左侧的查询，再右键新建查询

输入代码：select * from admin

即可查询admin这个表中的全部内容

select：选择来自哪个表（admin）

*：通配，即什么都匹配，将整个表的内容都显示出来

可以将*换成字段，若有多个字段，则用逗号隔开，如：

select username，password from admin

即可查询admin这个表中的username和password这两行内容

在表中添加内容：

若要只查询表中的admin这一行，可在后面加上where username=’admin’

即输入:select * from admin where username=’admin’

文本类型：html

页面编码：charset=utf-8（与数据库相同）

用变量来存储数据库的地址：127.0.0.1

数据库名：mybook

类与对象：new一个对象，即mysqli这个对象；server：数据库的地址；username：用户名；passwd：密码；dbname:所使用的mybook这个数据库
